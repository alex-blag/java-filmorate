
# Filmorate

The application allows users to rate movies.

## Features

- Movies moderation
- Movies search
- Movies rate

## Tech Stack

Spring Boot, H2

## Requirements

- Git
- Maven

## Run Locally

Clone the project

```
  git clone https://github.com/alex-blag/java-filmorate.git
```

Go to the project directory

```
  cd java-filmorate
```

Build and run project

```
  mvn spring-boot:run
```
